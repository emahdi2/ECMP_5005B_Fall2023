  <h1>📊 ECMP 5005 — Data Analytics</h1>

  <p>
    <strong>Course Audience:</strong> Master of Engineering (M.Eng.) and Data Science students
  </p>

  <p>
    ECMP 5005 is a graduate-level course in data analytics that introduces students to
    modern statistical learning and data-driven modeling techniques using the
    <strong>R programming language</strong>.
  </p>

  <p>
    The course is based on the textbook:
  </p>

  <blockquote>
    <em>An Introduction to Statistical Learning: With Applications in R</em> (2021)<br>
    Gareth James, Daniela Witten, Trevor Hastie, and Robert Tibshirani
  </blockquote>

  <p>
    The course emphasizes both theoretical understanding and practical implementation,
    with extensive hands-on use of R, RStudio, and R Markdown for reproducible data analysis.
  </p>

  <h3>🎯 Learning Objectives</h3>
  <p>By the end of this course, students will be able to:</p>
  <ul>
    <li>Use R, RStudio, and R Markdown for data analysis and reproducible reporting</li>
    <li>Explore, visualize, and summarize data using modern visualization tools</li>
    <li>Build, interpret, and evaluate regression and classification models</li>
    <li>Apply resampling techniques for model assessment and selection</li>
    <li>Implement regularization, dimension reduction, and ensemble learning methods</li>
    <li>Apply data analytics techniques to real-world engineering and data science problems</li>
  </ul>

  <h3>📚 Topics Covered</h3>
  <ol>
    <li>
      <strong>Introduction to R, RStudio, and R Markdown</strong>
      <ul>
        <li>R syntax and data structures</li>
        <li>Data import, cleaning, and manipulation</li>
        <li>Reproducible reporting with R Markdown</li>
      </ul>
    </li>
    <li>
      <strong>Data Visualization and Interactive Plots</strong>
      <ul>
        <li>Exploratory data analysis (EDA)</li>
        <li>Data visualization using <code>ggplot2</code></li>
        <li>Interactive plots using <code>plotly</code> and <code>shiny</code></li>
      </ul>
    </li>
    <li>
      <strong>Linear Regression</strong>
      <ul>
        <li>Simple and multiple linear regression models</li>
        <li>Model interpretation and diagnostics</li>
        <li>Prediction and inference</li>
      </ul>
    </li>
    <li>
      <strong>Resampling Methods</strong>
      <ul>
        <li>Training and test sets</li>
        <li>Cross-validation</li>
        <li>Bootstrap methods</li>
      </ul>
    </li>
    <li>
      <strong>Classification & Generalized Linear Models</strong>
      <ul>
        <li>Logistic regression</li>
        <li>Linear and quadratic discriminant analysis (LDA, QDA)</li>
        <li>Naive Bayes and k-nearest neighbors (KNN)</li>
        <li>Poisson regression</li>
      </ul>
    </li>
    <li>
      <strong>Linear Model Selection, Regularization & Dimension Reduction</strong>
      <ul>
        <li>Subset selection methods</li>
        <li>Ridge regression and lasso</li>
        <li>Principal components analysis (PCA)</li>
        <li>Factor analysis</li>
      </ul>
    </li>
    <li>
      <strong>Decision Trees and Ensemble Methods</strong>
      <ul>
        <li>Decision trees for regression and classification</li>
        <li>Bagging</li>
        <li>Random forests</li>
        <li>Boosting</li>
      </ul>
    </li>
  </ol>

  <h3>🛠 Software & Tools</h3>
  <ul>
    <li>R and RStudio</li>
    <li>R Markdown for reproducible reports</li>
    <li>Key R packages: <code>tidyverse</code>, <code>ggplot2</code>, <code>caret</code>, <code>glmnet</code>, <code>randomForest</code>, <code>e1071</code></li>
  </ul>

  <h3>📌 Notes</h3>
  <ul>
    <li>All lectures and labs are based on hands-on R coding and real datasets</li>
    <li>Students are expected to complete weekly labs and assignments</li>
    <li>No prior machine learning background is required, but basic statistics is assumed</li>
  </ul>
